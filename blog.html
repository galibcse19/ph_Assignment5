Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents an HTML or XML document as a structured tree of objects. It allows developers to manipulate the document's structure, style, and content using JavaScript.

How do you select an element from the DOM?

We can select an element from the DOM using methods like document.getElementById('id') or document.querySelector('.class'). These allow you to access and manipulate elements by ID, class, or tag.

What is event delegation in the context of the DOM and Why is it useful?

Event delegation is a technique where a single event listener on a parent element manages events for multiple child elements through event bubbling.Event delegation is useful because it enhances performance by reducing the number of event listeners attached to individual elements, saving memory.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes using methods like setAttribute('attrName', 'value') to set an attribute and getAttribute('attrName') to retrieve it. For styles, you can directly modify the style property, like element.style.color = 'red' to change text color. Additionally, you can use classList.add('className') or classList.remove('className') to manage CSS classes.
